# SOP-INC-001: Incident and Near-Miss Reporting Procedure

**Revision:** 5  |  **Effective Date:** 2019-03-01  |  **Owner:** EHS

## 1. Purpose
Defines the process for reporting, documenting, and following up on
incidents and near-misses.

## 2. Scope
Applies to all personnel plant-wide. A near-miss is any event that
could have resulted in injury, equipment damage, or environmental
release but did not.

## 3. Procedure
1. Report the event to your supervisor and EHS immediately, or within
   the shift for non-urgent near-misses.
2. Secure the area if there is any ongoing hazard.
3. EHS or designated investigator completes the incident report form,
   including root cause discussion where applicable.
4. Repeated near-misses on the same equipment or with the same root
   cause should be flagged for a lessons-learned review rather than
   handled as isolated events.
5. Corrective actions are tracked to closure and referenced in the
   equipment's maintenance history.

## 4. Notes
Near-misses are treated with the same reporting rigor as incidents
resulting in actual harm or damage - the goal is pattern detection
before a more serious event occurs.
