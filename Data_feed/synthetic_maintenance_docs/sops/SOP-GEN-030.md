# SOP-GEN-030: Emergency Generator Testing and Certification

**Revision:** 3  |  **Effective Date:** 2021-01-10  |  **Owner:** EHS / Facilities

## 1. Purpose
Defines testing and certification requirements for emergency backup
generators.

## 2. Scope
Applies to all emergency diesel generators, including GEN-04.

## 3. Testing Requirements
- Monthly: no-load run test, minimum 15 minutes, all gauges monitored.
- Annual: full load-bank test at 50%, 75%, and 100% rated load, 30
  minutes per step, plus automatic transfer switch operation check.

## 4. Procedure (Annual Certification)
1. Schedule load-bank test with operations and notify EHS.
2. Perform staged load test per Section 3, recording all readings.
3. Verify automatic transfer switch operates within spec transfer
   time.
4. Complete the annual test data sheet with all readings.
5. **Section 4 - Certification Sign-Off:** The facility EHS manager
   must review and sign the completed test data sheet to formally
   close the annual certification. The certification is not
   considered complete until this sign-off is on file, regardless of
   whether the test itself passed.
6. File signed certification with facility compliance records.

## 5. Notes
A generator that has passed all load-bank test steps but lacks the
EHS manager sign-off is NOT considered certified for compliance
purposes. This distinction should be clearly reflected in inspection
reports.
